    LiveDiag

    Live Preview App for Markdown+blockdiag(+seqdiag +actdiag +nwdiag)

    all block code from blockdiag.com. Thanks!!

    Platform

    • Mac OS 10.8+

    Requirement

    • blockdiag (add path to /usr/bin/blockdiag)
    • seqdiag (add path to /usr/bin/seqdiag)
    • actdiag (add path to /usr/bin/actdiag)

    • nwdiag (add path to /usr/bin/nwdiag)

      at this time cannot load user $PATH. I'm trying to improve it.

    Feature

    • live preview for following content
      • Markdown
      • blockdiag
      • seqdiag
      • actdiag
      • nwdiag
    • printing
